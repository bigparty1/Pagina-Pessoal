  • Sistema de Reservas de Hotéis — CLI com Arquitetura MVC ( Sistema_de_Hotelaria)

    Este projeto é uma aplicação de linha de comando (CLI) desenvolvida em JavaScript que simula um sistema de reservas de hotéis. Embora o escopo do problema seja simples, o diferencial está na implementação da arquitetura MVC (Model-View-Controller) — uma abordagem geralmente aplicada a aplicações web — adaptada aqui para um ambiente de terminal. O objetivo é explorar a separação de responsabilidades e aplicar boas práticas de organização de código mesmo em contextos minimalistas, como a interface de console. Este projeto serve como exercício de abstração e organização de código, visando evoluir minhas habilidades na jornada como desenvolvedor.
